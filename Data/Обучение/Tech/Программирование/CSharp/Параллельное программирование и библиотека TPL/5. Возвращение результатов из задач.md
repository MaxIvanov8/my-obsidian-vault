`int` `n1 =4, n2 = 5;`
`Task<int> sumTask =` `new` `Task<int>(() => Sum(n1, n2));`
`sumTask.Start();`
`int` `result = sumTask.Result;`
`Console.WriteLine($"{n1} + {n2} = {result}");` `// 4 + 5 = 9`

`int` `Sum(int` `a,` `int` `b) => a + b;`

Чтобы получать из задачи не который результат, необходимо типизировать объект Task тем типом, объект которого мы хотим получить из задачи.
Возвращаемое число будет храниться в свойстве Result: `sumTask.Result`.
При этом при обращении к свойству Result текущий поток останавливает выполнение и ждет, когда будет получен результат из выполняемой задачи.
