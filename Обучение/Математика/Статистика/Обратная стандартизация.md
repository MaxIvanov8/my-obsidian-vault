Выразим x из [[Статистические гипотезы|z-статистики]]
$$
xˉ=μ+z⋅SE
$$
Теперь мы можем выбрать критические значения для z-статистики, например, уже известные нам −1.96 и 1.96, подставить их и найти границы для среднего значения по выборке.

- Левая граница: μ−1.96⋅SE
- Правая граница: μ+1.96⋅SE.
![[Pasted image 20250109155144.png]]
Иногда границы обозначают как C (от англ. Сritical — критичный). Значения −C и C соответствуют пограничным значениям для среднего по выборке. Начиная с этих значений, мы считаем отклонения статистически значимыми. Если среднее значение по выборке окажется левее −C или правее C, то мы отвергнем H0​.
