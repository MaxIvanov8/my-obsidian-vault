### Обучение без учителя
Использование алгоритмов для поиска закономерностей.

### Обучение с учителем
Использование заданных шаблонов для прогнозирования. Предсказывает результат на основе существующих шаблонов.

### Обучение с подкреплением
Использование закономерностей в данных, постоянное улучшение прогнозирования по мере появления новых результатов.
